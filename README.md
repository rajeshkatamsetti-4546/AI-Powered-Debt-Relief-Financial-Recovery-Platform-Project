# AI Powered Debt Relief Financial Recovery Platform

## Project Overview
This project is an AI-powered debt relief financial recovery platform developed using FastAPI, Streamlit, and SQLite. It helps users manage loans, analyze financial health, and predict suitable debt settlement strategies.

## Features
- User Authentication
- Loan Management (Create, Read, Update, Delete)
- Financial Health Analysis
- Settlement Prediction
- AI-Based Negotiation Support

## Technologies Used
- Python
- FastAPI
- Streamlit
- SQLite
- SQLAlchemy
- Uvicorn

## Project Structure
```
backend/
frontend/
requirements.txt
README.md
```

## How to Run

### Backend
```bash
uvicorn app.main:app --reload
```

### Frontend
```bash
streamlit run frontend/app.py
```
